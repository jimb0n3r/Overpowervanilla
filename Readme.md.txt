# OPAlert - Vanilla WoW 1.12 Overpower Alert

Simple, clean addon that shows a pulsing Overpower icon when your target dodges.

## Features
- Clean square Overpower icon
- Nice pulsing glow effect
- Built-in + custom sound support (Fahh meme sound)
- Movable frame (drag with mouse)
- Remembers position

## Commands
- `/opalert test` — Test the alert
- `/opalert sound` — Toggle sound on/off
- `/opalert fahh` — Toggle between Fahh and built-in sound

## Installation
1. Download the latest release
2. Extract the `OPAlert` folder
3. Put it in `Interface\AddOns\`
4. Restart WoW or `/reload`

Enjoy those Overpowers!
